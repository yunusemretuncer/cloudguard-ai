# CloudGuard AI

Cloud Security Monitoring & Incident Response Agent — bulut güvenlik loglarını analiz eden, tehdit tespiti yapan ve müdahale önerileri sunan AI agent sistemi.

## Tech Stack
- **LLM:** Google Gemini 2.0 Flash
- **Agent:** LangChain
- **Backend:** FastAPI
- **Frontend:** React + Tailwind
- **DB:** SQLite
- **Deploy:** Docker Compose
